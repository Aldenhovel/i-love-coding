# TypeScript

存放 TypeScript 的相关代码仓库，因为不常用所以整理成笔记防止忘记。安装 TypeScript 需要保证：

- Node.js 安装

- TypeScript 安装

  ```shell
  npm install typescript -g
  ```

使用编译器将 `.ts` 文件自动编译为 `.js` 文件：

```shell
cd {path} 或在目录下打开终端
tsc {filename}.ts
```
